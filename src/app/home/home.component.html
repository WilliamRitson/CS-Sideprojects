Cache Simulator Status: Probably Stable

Cache Simulator which can simulate memory accesses displaying the result of each line of code and the status of the cache.

You can configure the cache size, set associativity, address size, addressing type, block size and replacement policy.

 Pipeline Diagram Generator Status: Experimental

Pipeline simulator which can generate pipeline diagrams from snippets of MIPS code (not all valid MIPS code will work).

You can activate or disable forwarding as well as configure the runtime of each instruction and what functional unit they use.

 Graph Search Status: Probably Stable

Visual Graph which can be used to run breadth or depth first search on directed or undirected graphs.

You can run each algorithm step by step and see how the colors and tables (parent, discovered, finished, ect) change over time.

 Minimax Status: Probably Stable

Demonstration of the minimax algorithm with tic-tac-toe

Eventually I would like to add other games and algorithms

 Stable Marriage Status: Experimental

Demo of the Gale Shapley stable marriage algorithm on random groups.

Eventually I may add a manual input method and other matching algorithms
